# ServiceContract — v2.1

**Schema Pack Version:** 2.1.0
**Released:** April 2026
**Status:** Initial release

## Notes

Initial release of the ServiceContract schema targeting the v2.1 generalised model. This schema captures contract-level metadata for a confirmed service booking, including the booking channel, client notes, and operational flags like reminder timestamps.

### Container

- **Contract.contractAttributes**

### Key features

- Multi-channel booking support (app, voice, in-person, SMS, WhatsApp, USSD)
- Client notes capture at booking time
- Cancellation reason tracking
- Facilitator identification for marketplace scenarios
- Reminder timestamp recording
