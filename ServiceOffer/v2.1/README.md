# ServiceOffer — v2.1

**Schema Pack Version:** 2.1.0
**Released:** April 2026
**Status:** Initial release

## Notes

Initial release of the ServiceOffer schema targeting the v2.1 generalised model. This schema defines offer-level terms and conditions for professional services, binding offers to resources with terms like quantity limits, advance booking requirements, and cancellation policies.

### Container

- **Offer.offerAttributes**

### Key features

- Quantity constraints (minimum and maximum)
- Advance booking requirements
- Flexible cancellation policies (time-based)
- Bulk discount eligibility
