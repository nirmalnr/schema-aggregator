# ServiceConsideration — v2.1

**Schema Pack Version:** 2.1.0
**Released:** April 2026
**Status:** Initial release

## Notes

Initial release of the ServiceConsideration schema targeting the v2.1 generalised model. This schema defines the pricing and payment terms for a service, including per-unit pricing, total amount, accepted payment methods, and transaction reference tracking.

### Container

- **Consideration.considerationAttributes**

### Key features

- Per-unit and total pricing
- Multi-currency support
- Multiple payment method support (cash, card, mobile, wallet, etc)
- Payment method selection tracking
- Transaction reference recording
