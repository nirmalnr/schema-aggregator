# ServicePerformance — v2.1

**Schema Pack Version:** 2.1.0
**Released:** April 2026
**Status:** Initial release

## Notes

Initial release of the ServicePerformance schema targeting the v2.1 generalised model. This schema captures the execution details of a confirmed service, including appointment timing, location, completion status, and follow-up scheduling.

### Container

- **Performance.performanceAttributes**

### Key features

- Appointment window and confirmed time tracking
- Service location with GPS coordinate support
- Provider and client completion flags
- Service notes from provider
- Follow-up scheduling
