# DisputeAction

DisputeAction is the container for the dispute / on_dispute messages exchanged between nodes. It carries the Dispute object being created, updated, status-tracked, or cancelled (withdrawn). The action being performed is conveyed by the endpoint (dispute / on_dispute) and the lifecycle state of the carried Dispute, rather than by a separate action code.
