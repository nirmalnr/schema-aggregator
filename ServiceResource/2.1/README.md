# ServiceResource — v2.1

**Schema Pack Version:** 2.1.0
**Released:** April 2026
**Status:** Initial release

## Notes

Initial release of the ServiceResource schema targeting the v2.1 generalised model. This schema defines the core resource attributes for on-demand professional services including teleconsultation, maintenance, repair, and consulting services.

### Container

- **Resource.resourceAttributes**

### Key features

- Flexible service unit types (hourly, session-based, per-visit, per-unit)
- Geographic coverage specification with radius support
- Weekly availability scheduling
- Capacity management (daily and weekly limits)
- Multi-language support
- Advance booking requirements
