# CommunicationChannel

A channel through which communication or support takes place. The enumerated values — `PHONE`, `EMAIL`, `WEB`, `CHAT`, `WHATSAPP`, `IN_APP`, `OTHER` — are derived from `SupportInfo.channels`.
